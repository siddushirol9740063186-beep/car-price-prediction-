🚗 Car Price Prediction
📌 Project Overview

This project uses Machine Learning to predict the price of used cars based on various features. It helps users estimate a fair price quickly and accurately.

📊 Dataset

The model is trained using the CarDekho dataset, which contains information about used cars such as:

Year of manufacture
Present price
Fuel type
Seller type
Transmission
Kilometers driven
⚙️ Technologies Used
Python
Pandas
NumPy
Scikit-learn
Streamlit
🚀 Features
🔹 Easy car price prediction
🔹 User-friendly interface
🔹 Fast and accurate results
🔹 Real-time prediction using Streamlit
